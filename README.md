# Vortex Wing

Pythonと [Ursina Engine](https://www.ursinaengine.org/) を使用して作成された、シンプルで爽快な3Dレールシューティングゲームです。

## 概要

プレイヤーは青い機体を操作し、無限に続くサイバー空間を前進しながら、立ちはだかる敵を撃破し、障害物を避けながらハイスコアを目指します。

## 必要条件（前提環境）

- Python 3.6以上
- `ursina` ライブラリ

## インストール手順

1. このリポジトリをクローンまたはダウンロードします。
2. 仮想環境を作成し、アクティベートします。
   ```powershell
   python -m venv .venv
   .venv\Scripts\Activate.ps1
   ```
3. 必要なライブラリをインストールします。
   ```powershell
   pip install ursina
   ```

## 遊び方（起動方法）

仮想環境が有効な状態で、以下のコマンドを実行してゲームを起動します。

```powershell
python vortex_wing.py
```

## 操作方法

| キー | アクション |
| :--- | :--- |
| **W, A, S, D** または **矢印キー** | 機体の移動（上下左右） |
| **スペースキー (Space)** | レーザー発射（攻撃） |
| **Q** または **E** | バレルロール（無敵回避アクション）※ Qで左回転、Eで右回転 |
| **R** | ゲームオーバー時にリスタート |

## ゲームシステム

- **自動前進**: 機体は自動的に奥（Z軸方向）へと進み続けます。
- **HP（体力）**: 初期HPは100です。敵の弾に当たったり、障害物にぶつかると減少します。0になるとゲームオーバーです。
- **無敵時間（バレルロール）**: `Q` または `E` キーで回転している間は無敵状態となり、敵の攻撃や障害物との衝突を無効化できます。
- **スコア**: 敵を倒すことでスコアが加算されます（1体につき100点）。生存時間が長くなるほど敵の出現頻度が上がります。
